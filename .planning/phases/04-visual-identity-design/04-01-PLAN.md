**GResource manifest** (`data/io.github.matrixshader.ShadowSettings.gresource.xml`):
- Prefix: `/io/github/matrixshader/ShadowSettings`
- Include: `style.css`, `style-dark.css`, `style-hc.css`, `style-hc-dark.css`

**GSettings schema** (`data/io.github.matrixshader.ShadowSettings.gschema.xml`):
- Schema ID: `io.github.matrixshader.ShadowSettings`
- Path: `/io/github/matrixshader/ShadowSettings/`
- Key `theme` (string, default `'auto'`): Visual theme selection (auto, gotham-night, gotham-day, wayne-manor)
- Key `reduce-motion` (boolean, default `false`): Disable animations

**CSS stylesheets** - Art Deco / Batman: The Animated Series aesthetic:

`data/style.css` (base, loaded in ALL modes):
- Define CSS variable overrides for all 3 themes via window class selectors:
  - `window.gotham-night`: deep blacks (#0A0A0F window bg), amber/gold accents (#C8962E primary, #D4A843 accent), warm parchment text (#D4C5A0), deep crimson for destructive (#A02020)
  - `window.gotham-day`: warm grays (#F0EDE6 window bg), muted gold accents (#9A7520), dark text
  - `window.wayne-manor`: deep browns (#1A1410 window bg), warm amber accents (#C89A3E), warm text
- Art Deco styling rules (apply to ALL themes):
  - Monospace, uppercase, letter-spaced group title headers with accent color (Batcomputer feel)
  - Art Deco divider borders on preference group headers (2px solid accent at ~40% opacity)
  - Sidebar pinstripe: subtle gradient border-right on sidebar pane
  - `setting-modified` class: inset left border in amber (persistent indicator, no animation yet -- Plan 02 adds animation)
  - Row styling: subtle card-like backgrounds on preference rows
  - Scanline texture on window background (repeating-linear-gradient at very low opacity, 2px lines)
  - HUD-style category headers if it looks good (Claude's discretion on depth)

`data/style-dark.css` (loaded ONLY in dark mode, stacks on top of style.css):
- Can be minimal or empty if all dark handling is via CSS classes
- Include any dark-mode-specific adjustments not covered by class selectors

`data/style-hc.css` (high contrast mode):
- Ensure all text meets WCAG AAA contrast ratios
- Thicken borders, increase accent opacity
- Override any subtle effects that reduce readability

`data/style-hc-dark.css` (high contrast + dark):
- Combine HC and dark adjustments
- Ensure modified-setting indicators are clearly visible

Create the ThemeManager class and integrate a Preferences panel as the first sidebar item:

**ThemeManager** (`src/core/theme-manager.vala`):
- Namespace: `ShadowSettings`
- Public class `ThemeManager : Object`
- Constructor takes `Adw.Application` reference
- Private field: `GLib.Settings app_settings` bound to `io.github.matrixshader.ShadowSettings` schema (use `SafeSettings.try_get()` to null-guard -- if schema not found, default to auto/gotham-night)
- Private field: `Adw.StyleManager style_manager` = `Adw.StyleManager.get_default()`
- Public method `apply_theme(string theme_id)`:
  - Remove all theme CSS classes from active window (`gotham-night`, `gotham-day`, `wayne-manor`)
  - Switch on theme_id:
    - `"gotham-night"`: `FORCE_DARK`, add class `gotham-night`
    - `"gotham-day"`: `FORCE_LIGHT`, add class `gotham-day`
    - `"wayne-manor"`: `FORCE_DARK`, add class `wayne-manor`
    - `"auto"` (default): `ColorScheme.DEFAULT`, then check `style_manager.dark` -- if dark add `gotham-night`, else add `gotham-day`
  - Persist to GSettings `theme` key
- Public method `set_reduce_motion(bool reduce)`:
  - Add/remove `reduce-motion` CSS class on active window
  - Set `gtk_enable_animations` on `Gtk.Settings.get_default()` accordingly
  - Persist to GSettings `reduce-motion` key
- Public property `string current_theme` (getter from GSettings)
- Public property `bool reduce_motion_active` (getter from GSettings)
- On construction: read persisted theme from GSettings, call `apply_theme()` with it
- Listen to `style_manager.notify["dark"]` -- if theme is "auto", re-apply to switch between gotham-night/gotham-day when system changes
- NOTE: Must handle the case where SafeSettings.try_get() returns null for the app schema (first run before schema install) -- default to "auto" theme and gotham-night/gotham-day based on system dark preference

**Preferences panel in window.vala:**
